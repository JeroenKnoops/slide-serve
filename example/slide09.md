# Styling with CSS

# Selectors
..slide
..slide h1
..slide strong
..slide pre
.
..slide-N, where N is slide number

# Example
.
  .slide-10 h1 {
    color: #fff;
    text-shadow: 1px 1px 3px #453029;
  }
.

# CSS backgrounds
  .slide-12 {
    background: url("icon.png"),
                url("bg.png");
    background-repeat: no-repeat, repeat;
    background-size: 40%, auto;
    background-position: bottom right;
  }
